Installation

lumX is the first responsive front-end framework based on AngularJS and Google Material Design specifications.
It provides a CSS Framework built with Sass and a bunch of AngularJS directives.

Dependencies

lumX is available through Bower:
bower install lumx

Donâ€™t want bower? You first need to install lumX dependencies :

  • AngularJS
  • jQuery
  • Velocity
  • Bourbon
  • Neat

Why these dependencies?

Our CSS framework is based on bourbon, a simple and lightweight mixin library for Sass:
http://bourbon.io

Neat is a lightweight semantic grid framework for Sass and Bourbon, used to build our grid system:
http://neat.bourbon.io

Velocity is used to provide smooth animations for the dropdown directive:
http://julian.com/research/velocity

Use lumX in your project

When you have downloaded all the dependencies you need to add dependencies on the lumX AngularJS module:
angular.module('myModule', ['lumx']);

Then you juste have to link to the main css file (dist/css/lumx.css).
